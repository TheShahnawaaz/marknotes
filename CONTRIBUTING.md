# Contributing to MarkNotes

Thanks for your interest in contributing! 

## Getting Started

1. Fork the repo
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/marknotes.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Test: `npm run tauri dev`
6. Commit: `git commit -m "Add your feature"`
7. Push: `git push origin feature/your-feature`
8. Open a Pull Request

## Development Setup

See [README.md](README.md#development) for prerequisites and setup instructions.

## Code Style

- TypeScript for frontend
- Rust for backend
- Follow existing code patterns
- Run `npm run build` before committing to ensure it compiles

## Pull Request Guidelines

- Keep PRs focused on a single feature or fix
- Update README if you add features
- Test on both macOS and Windows if possible
- Describe what you changed and why

## Questions?

Open an issue or discussion on GitHub.
